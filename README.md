# ScrapRace
3DGame Development

操作説明  
メニュー画面  
Wキー、上キー・・・上に移動  
Sキー、下キー・・・下に移動  
エンターキー・・・決定ボタン  

プレイ画面  
WASDキー、または矢印キーでキーで移動・方向転換  
Wキー、上キー・・・前方向へ移動  
Sキー、下キー・・・後ろへ移動  
Aキー、左キー・・・左へ旋回  
Dキー、右キー・・・右へ旋回  

リザルト画面  
Aキー、左キー・・・左へ移動  
Dキー、右キー・・・右へ移動  
エンターキー・・・決定ボタン  

遊び方  
一位を目指してコースを周回してゴールしましょう  
時間経過で体力が減っていくのでコースに落ちているスクラップ（赤い球体）、または敵から落ちるレアスクラップ（緑の球体）を拾って回復しましょう  
スクラップは体力回復の他に、最高速度、加速度がランダムで上昇します。最高速度、加速度は時間経過で減っていくので多くとった方が有利です。
